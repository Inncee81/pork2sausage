This is a demo program made in java for Pork2Sausage (un Notepad++ plugin).
The program zip or unzip a given string from stdin then put the result to stdout.

Example to zip:
java -classpath C:/pork2Sausage/zip.base64;C:/pork2Sausage/zip.base64/commons-codec-1.4.jar zipB64 -zip "eNoLycgsVgCikoxUhZLUihKFknyFpFSF5PzcgqLU4uLUFADH7gwa"

Example to unzip:
java -classpath C:/pork2Sausage/zip.base64;C:/pork2Sausage/zip.base64/commons-codec-1.4.jar zipB64 -unzip eNoLycgsVgCikoxUhZLUihKFknyFpFSF5PzcgqLU4uLUFADH7gwa
